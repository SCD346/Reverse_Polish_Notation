Reverse Polish Notation

Description: Write a program to function as a Reverse Polish notation calculator.

Here are some examples of inputs and outputs from this program:

2 5 +
# equivalent to: 2 + 5 = 7
# output: 7
1 30 -
# equivalent to: 1 - 30 = -29
# output: -29
3 6 * 2 /
# equivalent to: (3 * 6) / 2 = 9
# output: 9
3 4 2 * -
# equivalent to: 3 - (4 * 2) = -5
# output: -5
10 9 8 7 - + +
# equivalent to: 10 + (9 + (8 - 7)) = 20
# output: 20
1
# output: 1

Note: For the sake of simplicity, for now you can ignore invalid input strings and assume that all input numbers are integers
